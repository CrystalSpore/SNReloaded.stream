Today, we will be talking about Linux! Specifically a look at it from a beginner's perspective. The two versions of Linux we will be looking at is Ubuntu (both desktop and server versions), and Elementary OS.

Ubuntu is my daily driver. It's what I use on both my personal computer (using Ubuntu desktop), and on my server computer (which runs this website that you are currently viewing, using Ubuntu server). At this point you might be thinking, what's the big difference between the desktop and server versions of Ubuntu? The main difference is that the desktop version has a desktop environment, while the server version does not. A desktop environment is basically what you would be used to seeing in either Windows or MacOS.

Ubuntu is fairly widespread, due to it's relative ease to set-up, easy desktop environment with it's similarities to Windows/MacOS, and deep knowledge base, while also having the whole linux backend with the terminal to be able to configure things at an advanced level if you need/want to.
