This page is a list of commands that I commonly use when using linux (I use a Debian based system, but most of these commands will still be the same). Hopefully these commands will help you improve your efficiency with linux!

Command Description/Use
sudo Used to elevate a command to root level. Windows equivalent would be running something as Adinistrator
sudo -i OR sudo bash Are you needing to run a lot of root level commands? Just use one of these commands and it will raise you to root level. (I personally use sudo bash for clarity)
apt Used on Debian based systems (such as Ubuntu) to install applications/utilities from the terminal. Typically needs a root level permission.
